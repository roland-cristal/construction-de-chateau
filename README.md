# 🏰 Construction de Château

Ce dépôt est un hommage non officiel à l’application **“Construction de Château”** proposée par Roland Cristal.

Une œuvre interactive, ludique et singulière permettant de composer librement des châteaux à partir d’éléments graphiques.

Le logiciel est disponible à l'adresse : https://roland-cristal.github.io/construction-de-chateau/

## 🏰 Le projet

“Construction de Château” est un éditeur visuel interactif permettant de :

- Ajouter tours, murs, fenêtres, ponts et éléments architecturaux
- Redimensionner les éléments librement
- Modifier leur ordre d’affichage
- Changer de fond
- Activer un **mode Fiesta synchronisé au BPM**

L’utilisateur construit son propre château par superposition et manipulation directe.

L’interface volontairement minimaliste et rétro (inspiration Paint / Windows 95) renforce l’aspect brut et ludique du projet.

## 🎮 Commandes

### 🖱 Souris
- **Molette** → Agrandir / réduire
- **Molette + V** → Agrandir / réduire verticalement
- **Molette + H** → Agrandir / réduire horizontalement
- **Clic droit** → Supprimer un élément
- **Glisser-déposer** → Déplacer un élément

### ⌨ Clavier
- **+** → Mettre au premier plan
- **-** → Mettre au dernier plan

### 🎛 Mode Fiesta
- Entrer un BPM
- Cliquer sur **Fiesta**
- Les éléments se déforment et rebondissent en rythme


## 🌟 Pourquoi ce dépôt ?

Parce que cette application mérite d’être partagée.  
Parce qu’un éditeur de château avec un mode Fiesta BPM est une idée trop belle pour disparaître.
